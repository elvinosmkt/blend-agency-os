# 🛡️ Cybersecurity Squad

15 agents orchestrated by the Cyber Chief. Authorization-first framework.

## Agents
- 🛡️ **Cyber Chief** — Orchestrator. Verifies auth, plans operations, routes specialists.
- **Chris Sanders** — Blue team, network security monitoring, packet analysis.
- **Georgia Weidman** — Mobile pentesting, penetration testing methodology.
- **Jim Manico** — AppSec, OWASP, secure coding, web application security.
- **Marcus Carey** — Threat intelligence, red team strategy, breach simulation.
- **Omar Santos** — Network security, vulnerability assessment, incident response.
- **Peter Kim** — Red team, Hacker Playbook methodology, advanced exploitation.
- 🗺️ **Cartographer** — Recon, attack surface mapping, network topology.
- 🔍 **Busterer** — Web content discovery, directory enumeration, endpoint mapping.
- 📂 **Dirber** — Network service enumeration (SMB, LDAP, SNMP, AD).
- 🎯 **Fuzzer** — Input fuzzing, parameter manipulation, injection testing.
- 🔓 **Ripper** — Credential cracking, password security assessment.
- 💀 **Rogue** — Exploitation, post-exploitation, privilege escalation.
- 🔎 **Shannon Runner** — OSINT collection, social engineering recon.
- ⚡ **Command Generator** — Precise tool command syntax generation.

## Operation Flows
- **Full Pentest:** auth check → cartographer → dirber/busterer → fuzzer → rogue → peter-kim
- **Web AppSec:** jim-manico → busterer → fuzzer → command-generator
- **Incident Response:** omar-santos → chris-sanders → marcus-carey
- **OSINT:** shannon-runner → cartographer → marcus-carey

⚠️ Authorization REQUIRED before any offensive operation.
