# 🏛️ Advisory Board

11 world-class advisors orchestrated by the Board Chair. Each brings a distinct worldview.

## Agents
- 🏛️ **Board Chair** — Orchestrator. Routes questions, manages productive tension, synthesizes counsel.
- 📐 **Ray Dalio** — Principles, decision systems, economic cycles, radical transparency.
- 🧠 **Charlie Munger** — Mental models, inversion, cognitive bias, multidisciplinary thinking.
- 🧘 **Naval Ravikant** — Wealth, leverage, specific knowledge, angel investing, happiness.
- ⚡ **Peter Thiel** — Contrarian thinking, monopoly, Zero to One, power law.
- 🌐 **Reid Hoffman** — Blitzscaling, network effects, ABZ planning.
- 🎯 **Simon Sinek** — Start With Why, Golden Circle, Infinite Game.
- 💛 **Brené Brown** — Vulnerability, courage, trust, wholehearted leadership.
- 📋 **Patrick Lencioni** — Five Dysfunctions, team health, organizational health.
- 🎯 **Derek Sivers** — Hell Yeah or No, minimalist entrepreneurship.
- 🏔️ **Yvon Chouinard** — Mission-driven, sustainability, purpose over profit.

## Panels
- **Investment Committee:** Dalio + Munger + Naval
- **Scaling Council:** Hoffman + Thiel + Sivers
- **Culture Circle:** Lencioni + Brown + Sinek
- **Founder Council:** Naval + Sivers + Chouinard
