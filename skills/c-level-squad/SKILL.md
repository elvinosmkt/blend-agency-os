# 👔 C-Level Squad

Virtual C-suite orchestrated by the Vision Chief (CEO).

## Agents
- 👔 **Vision Chief (CEO)** — Vision, strategy, fundraising, culture, board, M&A, pivots.
- ⚙️ **COO Orchestrator** — Operations, scaling, process, team structure, OKRs.
- 📣 **CMO Architect** — Brand positioning, go-to-market, demand generation.
- 🔧 **CTO Architect** — Technology strategy, architecture, engineering culture.
- 🖥️ **CIO Engineer** — Information systems, security, compliance, IT governance.
- 🤖 **CAIO Architect** — AI strategy, ML pipelines, LLM integration, responsible AI.

## Routing
| Challenge | Route To |
|-----------|----------|
| Vision / strategy / culture / board / M&A | Vision Chief |
| Operations / scaling / OKRs | COO Orchestrator |
| Marketing / brand / GTM | CMO Architect |
| Tech stack / architecture | CTO Architect |
| Security / compliance / IT | CIO Engineer |
| AI adoption / LLM / automation | CAIO Architect |
